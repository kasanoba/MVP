📋 프로젝트 개요

🎯 목적

금융권 및 대기업의 레거시 COBOL 시스템을 현대적인 프로그래밍 언어(Python, Java)로 효율적으로 변환하여 시스템 현대화를 지원합니다.

⚡ 효율성
수동 변환 대비 90% 시간 단축
💡 지능성
AI 기반 정확한 코드 변환
📊 투명성
변환 이력 및 비용 추적
🔒 안정성
일괄 처리 및 오류 관리

⚙️ 주요 기능

🔄 언어전환

  • COBOL → Python/Java 변환
  • 직접 입력 또는 파일 업로드
  • 기존 변환 이력 조회 및 재사용
  • API 호출 최적화로 비용 절감

📚 변환 이력 조회

  • 과거 변환 내역 조회
  • 로컬/클라우드 저장소 지원
  • 변환 유형별 필터링
  • 상세 변환 정보 제공

📦 일괄전환

  • 디렉토리 전체 파일 일괄 변환
  • ZIP 파일 업로드/다운로드
  • 실시간 진행률 표시
  • 예상 비용 및 토큰 수 안내

🏛️ 금융권 규정 검토

  • 금융권 코딩 규정 준수 검토
  • 보안 가이드라인 검증
  • 컴플라이언스 체크

🎯 선택 이유

Streamlit

  • 빠른 프로토타이핑
  • Python 개발자 친화적
  • 풍부한 UI 컴포넌트

OpenAI GPT-4o-mini

  • 높은 코드 변환 정확도
  • 비용 효율적
  • 다양한 언어 지원

👥 사용자 플로우

1단계: 🚀 앱 접속
환경변수 확인 및 Azure Blob 컨테이너 자동 생성
2단계: 📑 기능 선택
언어전환 / 변환 이력 조회 / 일괄전환 중 선택
3단계: 📝 입력
COBOL 코드 직접 입력 또는 파일/ZIP 업로드
4단계: 🔍 이력 확인
기존 변환 결과 조회 및 재사용 여부 결정
5단계: ⚡ 변환 실행
AI 기반 코드 변환 수행 (진행률 표시)
6단계: 📊 결과 확인
변환된 코드 검토 및 다운로드

🚀 실행 방법

1️⃣ 환경 설정

# 패키지 설치 pip install -r requirements.txt # 환경변수 설정 (.env 파일) OPENAI_API_KEY=sk-xxxxxxx OPENAI_ENDPOINT=https://api.openai.com/v1 CHAT_MODEL=gpt-4o-mini AZURE_STORAGE_CONNECTION_STRING=your_connection_string CONTAINER_NAME=your_container_name

2️⃣ 애플리케이션 실행

# Streamlit 앱 실행 streamlit run run_app.py # 또는 스크립트 사용 ./streamlit.sh

3️⃣ 접속 정보
